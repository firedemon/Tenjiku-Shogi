Tenjiku-Shogi by firedemon

Playing and analysing tool for Tenjiku Shogi

The program allows you to play and analyse Tenjiku Shogi games. You can show possible moves, influences, load and save games and positions. Several game variants are implemented. Output is done on an ANSI terminal, you can choose between, kanji, full kanji and ASCII. Graphic export can be done via LaTeX.

Dependencies

- glibc - GNU readline - ncurses - TeX for graphic export

Install

"make" should do it on a decently complete linux installation. Sorry, cannot help you for other operating systems, but please tell me if you manage to compile and run it.
